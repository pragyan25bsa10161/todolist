## üöÄ Project Overview

This project is a basic, menu-driven **To-Do List application** built entirely in **Python**. It's designed to run in the command-line interface (CLI) and uses a fundamental Python list (`tasks`) to store all the user's tasks. This assignment demonstrates core programming concepts such as functions, loops, conditional logic, and list manipulation.

---

## ‚ú® Features

The application provides the following four main functionalities:

* **View Tasks:** Display all current to-do items with their corresponding list number.
* **Add Task:** Prompt the user for a new task description and append it to the list.
* **Delete Task:** Allow the user to remove a task by entering its list number. Includes robust **input validation** to prevent errors.
* **Exit:** Terminates the application gracefully.
